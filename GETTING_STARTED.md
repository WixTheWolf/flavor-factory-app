# Getting Started Guide

Welcome to Flavor Factory B2B Mobile App! This guide will help you get started in 5 minutes.

## Quick Start

### 1. Install Dependencies

```bash
npm install
```

### 2. Start Development Server

```bash
npm start
```

### 3. Run on Device

**Android:**
```bash
npm run android
```

**iOS:**
```bash
npm run ios
```

## First Steps

### 1. Browse Products

- Navigate to the "Shop" tab
- Browse the product catalog
- Search for specific items
- Tap a product to view details

### 2. Create a Purchase Order

- View product details
- Adjust quantity (must meet MOQ)
- Tap "Add to Purchase Order"
- Items are saved to your cart

### 3. Checkout Your Order

- Go to "Cart" tab
- Review added items
- Tap "Complete Purchase Order"
- Fill in company and shipping info
- Submit your order

### 4. Track Orders

- Go to "Profile" → "My Orders"
- View all submitted purchase orders
- Check status: pending, confirmed, shipped, delivered
- View order details and history

### 5. Manage Your Profile

- Go to "Profile" tab
- Edit company information
- Update contact details
- View billing and order history

## Common Tasks

### How to Add Items to Your Order

1. Tap "Shop" tab
2. Browse or search for products
3. Tap a product card to view details
4. Increase quantity above MOQ
5. Tap "Add to Purchase Order"
6. Repeat for other items

### How to View Your Cart

1. Tap "Cart" tab
2. See all items in your current order draft
3. Adjust quantities or remove items
4. View estimated total

### How to Submit an Order

1. Go to "Cart" tab with items added
2. Tap "Complete Purchase Order"
3. Enter company information
4. Enter shipping address
5. Add PO number and notes (optional)
6. Review order summary
7. Tap "Submit Purchase Order"
8. Confirmation will be sent to your email

### How to Track Your Orders

1. Tap "Profile" tab
2. Tap "My Purchase Orders"
3. See list of all orders with status
4. Tap an order to view details
5. Statuses: Pending → Confirmed → Shipped → Delivered

## Pricing Example

**Product:** Vanilla Ice Cream (Bulk)
- Unit Price: $2.99 per lb (for 1-49 lbs)
- Bulk Price: $2.49 per lb (for 50+ lbs)
- MOQ: 50 lbs

**Scenarios:**
- Order 30 lbs: $2.99 × 30 = **$89.70** ⚠️ Below MOQ
- Order 50 lbs: $2.49 × 50 = **$124.50** ✓ Bulk pricing applied
- Order 100 lbs: $2.49 × 100 = **$249.00** ✓ Best value
