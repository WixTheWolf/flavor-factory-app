# Flavor Factory B2B Mobile App

A professional React Native mobile application for flavor ingredient ordering and purchase order management.

## Overview

Flavor Factory B2B is a specialized flavor ingredient ordering platform designed for manufacturers and businesses in bakery, beverage, pharmaceutical, nutraceutical, oral care, vape, confectionery, and dairy industries. The app provides seamless bulk purchasing with tiered pricing, technical specifications, application guidance, and full purchase order management integrated with vDosPlus inventory system.

## Key Features

- **Industry-Specific Flavor Catalog**: Browse 500+ flavors organized by application (Bakery, Beverage, Pharma, Nutraceutical, Oral Care, Vape, Confectionery, Dairy, Syrups)
- **Technical Information**: Full specifications, certifications (USP/FCC), and application suggestions
- **Tiered Pricing**: Unit pricing and bulk discounts based on Minimum Order Quantities
- **Purchase Order Management**: Draft, submit, and track POs with vDosPlus integration
- **Real-Time Inventory**: Stock status synchronized with vDosPlus system
- **Business Accounts**: Company profiles, shipping addresses, and order history
- **Direct Sales Support**: Ready for email, phone, and direct sales workflows

## Quick Start

```bash
npm install
npm start
npm run android   # or npm run ios
```

## Technology Stack

- **React Native 0.72** with TypeScript
- **Redux** for state management
- **React Navigation** for app structure
- **AsyncStorage** for local data
- **Axios** for API communication

## Integrations

- **vDosPlus**: Real-time inventory synchronization
- **Direct Sales CRM**: Support for email/phone order workflows
- **B2B Order Management**: Full purchase order lifecycle
